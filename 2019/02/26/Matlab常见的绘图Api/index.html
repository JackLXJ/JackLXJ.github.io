一般使用两种方式：

  • mesh: 可画出立体的网状图,即由多条曲线构成
  • surf：可画出立体的曲面图,即由多条曲面构成

为了方便测试立体绘图，MATLAB提供了一个peaks函数，可产生一个凹凸有致的曲面，包含了三个局部极大点（Local Maxima）及三个局部极小点（Local Minima）
